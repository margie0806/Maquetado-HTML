Himno del SENA

CORO
Estudiantes del SENA adelante
Por Colombia luchad con amor
Con el ánimo noble y radiante
Transformémosla en mundo mejor

I
De la patria el futuro destino,
en las manos del joven está,
el trabajo es seguro camino,
que el progreso a Colombia dará.

II
En la forja del SENA se forman,
hombres libres que anhelan triunfar,
con la ciencia y la técnica unidas,
nuevos rumbos de paz trazarán.

III
Hoy la patria nos grita sentida,
¡estudiantes del SENA triunfad!
solo así lograréis en la vida,
más justicia, mayor libertad.

IV
Avancemos con fuerza guerrera,
¡estudiantes con firme tesón!
que la patria en nosotros espera,
su pacífica revolución.

Glosario de Términos HTML

Glosario de Términos HTML

  • HTML: HyperText Markup Language, utilizado para crear páginas web.
  • Etiqueta: Elemento utilizado para definir la estructura y contenido de una página web.
  • Head: Sección de un documento HTML que contiene información sobre el documento.
  • Body: Sección de un documento HTML que contiene el contenido visible de la página web.
  • Título: Elemento utilizado para definir el título de una página web.
  • Meta: Elemento utilizado para proporcionar metadatos sobre el documento HTML.
  • Párrafo: Elemento utilizado para definir un párrafo de texto.
  • Coro: Sección de una canción que se repite después de cada estrofa.
  • Estrofa: Sección de una canción que contiene un conjunto de líneas con un tema común.
  • Enlace: Elemento utilizado para crear hipervínculos a otras páginas web.
